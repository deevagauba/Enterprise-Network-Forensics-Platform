# Enterprise Network Forensics Platform

Welcome to the Enterprise Network Forensics Platform repository.

## Overview
This platform is designed for network forensics, traffic simulation, and intrusion detection. It consists of a modern React frontend and a robust Python backend.

## Project Structure
- `backend/`: Python backend, detection engine, and traffic simulator.
- `frontend/`: React + Vite frontend for visualizing network topology and logs.
- `data/`: Sample data files, forensic logs, and network state configurations.

## Getting Started
To run the platform, use the provided `start_app.bat` script or start the backend and frontend separately.
